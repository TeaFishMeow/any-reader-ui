1. 应用斯托克斯公式计算曲线积分：

1) $\oint_{L} (z - y) \mathrm{d}x + (x - z) \mathrm{d}y + (y - x) \mathrm{d}z, L$ : 以 $A(a,0,0), B(0,a,0), C(0,0,a)$ 为顶点的三角形沿 $ABCA$ 的方向.

2) $\oint_{L}(y^{2} + z^{2})\mathrm{d}x + (x^{2} + z^{2})\mathrm{d}y + (x^{2} + y^{2})\mathrm{d}z,\quad L$ ：平面 $x + y + z = 1$ 与三坐标面的交线，其走向是使所围平面区域的上侧在曲线左方.

2. 验证所给表示式是全微分，并求原函数：

1) $yz\mathrm{d}x + xz\mathrm{d}y + xy\mathrm{d}z;$   
2) $(2xz + y)\mathrm{d}x + (x + z)\mathrm{d}y + (x^{2} + y)\mathrm{d}z$

3. 验证所给积分与路径无关，并求其值：

1) $\int_{(1,1,1)}^{(2,3,-4)} x\mathrm{d}x + y^2\mathrm{d}y + z^3\mathrm{d}z;$   
2) $\int_{(x_1,y_1,z_1)}^{(x_2,y_2,z_2)}\frac{x\mathrm{d}x + y\mathrm{d}y + z\mathrm{d}z}{\sqrt{x^2 + y^2 + z^2}}.$
