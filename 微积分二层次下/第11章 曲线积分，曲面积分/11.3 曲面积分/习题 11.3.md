1. 计算第一型曲面积分：

1) $\iint_{\Sigma} (3x + 6y + 4z) \, \mathrm{d}S,$ $\Sigma$ : 平面 $\frac{x}{4} + \frac{y}{2} + \frac{z}{3} = 1$ 在第一卦限的部分;  
2) $\iint_{\Sigma} \frac{1}{(1 + x + y)^2} \mathrm{d}S,$ $\Sigma: x + y + z = 1$ 及三个坐标面所围成的四面体的表面;   
3) $\iint_{\Sigma} (1 - x^2 - y^2) \, \mathrm{d}S,$ $\Sigma$ : 球面 $x^2 + y^2 + z^2 = 1$ ;  
4) $\iint_{\Sigma} \sqrt{a^2 - x^2 - y^2} \, \mathrm{d}S,$ $\Sigma$ : 上半球面 $z = \sqrt{a^2 - x^2 - y^2} (a > 0)$ ;  
5) $\iint_{\Sigma} (x^2 + y^2) \, \mathrm{d}S,$ $\Sigma$ : 曲面 $z = \sqrt{x^2 + y^2}$ 及平面 $z = 1$ 所围立体的表面;  
6) $\iint_{\Sigma} (xy + yz + zx) \, \mathrm{d}S,$ $\Sigma$ : 曲面 $z = \sqrt{x^2 + y^2}$ 被曲面 $x^2 + y^2 = 2ax$ ( $a > 0$ ) 所截得的部分;  
7) $\iint_{\Sigma} |xyz| \, \mathrm{d}S,$ $\Sigma: z = x^2 + y^2$ 被平面 $z = 1$ 割下的部分.

2. 计算第二型曲面积分：

1) $\iint_{\Sigma} (x + 2y + 3z) \, \mathrm{d}x \, \mathrm{d}y,$ $\Sigma$ : 平面 $x + y + z = 1$ 在第一卦限的上侧;  
2) $\iint_{\Sigma} \frac{\mathrm{e}^z}{\sqrt{x^2 + y^2}} \, \mathrm{d}x \, \mathrm{d}y,$ $\Sigma$ : 锥面 $z = \sqrt{x^2 + y^2}$ 及平面 $z = 1$ , $z = 2$ 所围成立体的外表面;  
3) $\iint_{\Sigma} (x + 1) \, \mathrm{d}y \, \mathrm{d}z + y \, \mathrm{d}z \, \mathrm{d}x + \mathrm{d}x \, \mathrm{d}y,$ $\Sigma$ : 平面 $x + y + z = 1$ 与三个坐标面所围的四面体的外表面;  
4) $\iint_{\Sigma} x \, \mathrm{d}y \, \mathrm{d}z + y \, \mathrm{d}z \, \mathrm{d}x + z \, \mathrm{d}x \, \mathrm{d}y,$ $\Sigma$ : 圆柱面 $x^2 + y^2 = a^2$ 界于 $z = h$ 与 $z = -h$ 之间的部分的外表面 $(a > 0, h > 0)$ ;  
5) $\iint_{\Sigma} yz \, \mathrm{d}x \, \mathrm{d}y + zx \, \mathrm{d}y \, \mathrm{d}z + xy \, \mathrm{d}z \, \mathrm{d}x,$ $\Sigma: x^2 + y^2 = R^2$ 与平面 $x = 0, y = 0, z = 0$ 及 $z = h (h > 0)$ 所围在第一卦限的立体的外表面.
