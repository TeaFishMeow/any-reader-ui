## 试卷

### 一、简答题（每小题6 分，共 42 分）

（1）．求解下列三元一次方程组，并指出其解的几何意义：

$$
\left\{ \begin{array}{c} x - y + 2 z = 6 \\ 3 x + y - z = 0 \\ y + z = 1 \end{array} \right.

$$

（2）. 设空间中四个点的坐标依次为 $A(0,-1,2),\ B(1,2,4),\ C(3,4,7),\ D(2,1,5)$。试证 $ABDC$ 是一个平行四边形，并计算该平行四边形的面积。

（3）．设 $u = e^{xy} \sin(x^2 + z^2)$，求 $\partial u/\partial x,\ \partial u/\partial y,\ \partial^2 u/\partial y\partial z$。

（4）．设隐函数 $z = f(x, y)$ 由 $xy + yz + z^3 = 3$ 确定，求该函数在 $(1, 1, 1)$ 处的全微分。

（5）．讨论二元函数 $z = \begin{cases} \dfrac{x^2 y^3}{x^4 + y^6}, & (x,y) \neq (0,0) \\ 0, & (x,y) = (0,0) \end{cases}$ 在 $(0,0)$ 处的连续性。

（6）.求过直线 $\begin{cases} x - y - 2 = 0 \\ 2y + z + 3 = 0 \end{cases}$ 且与球面 $x^2 + y^2 + (z-1)^2 = 2$ 相切的平面的方程。

（7）. 求函数 $f(x, y) = 4x^2 - 2xy + y^2 - 6x - 3$ 的极值点及相应的极值，并指出是极大值或极小值。

二、（8 分）设 $z = \ln(u + v^2)$，$u = e^x \cos y$，$v = e^x \sin y$，$w = xy - z$。试求 $\partial z/\partial x$ 及 $\partial z/\partial y$。

三、（10 分）考虑圆锥体 $\sqrt{x^2 + y^2} \leq z \leq 2$ 被椭圆柱面 $(x-1)^2 + 2y^2 = 1$ 截出的立体（即位于椭圆柱面内的部分），试画出该立体的草图，并分别求出：(1) 该立体在 $xOy$ 平面的投影；(2) 二曲面的交线在 $zOx$ 平面的投影。

四、（10 分）试求曲面 $x^2 + 2yz = 1$ 与直线 $\dfrac{x-2}{0} = \dfrac{y-1}{1} = \dfrac{z-1}{1}$ 距离最近的点，并计算这个距离。

五、（10 分）设区域 $D$ 是由 $x^2 + y^2 = 1,\ x^2 + y^2 = 4,\ y = 0,\ y = x$ 所围成的位于第一象限内的区域，试计算二重积分 $I = \iint_D \frac{\sin\sqrt{x^2+y^2}}{\sqrt{x^2+y^2}}\, dx\, dy$。

六、（12 分）已知椭球面 $S_1$ 的方程为 $x^2 + 2y^2 + z^2 = 2$，锥面 $S_2$ 的方程为 $z = \sqrt{x^2 + y^2}$，试分别求出点 $(1, 0, 1)$ 处 $S_1$ 的切平面与 $S_2$ 的切平面的方程，并求出 $S_1$ 与 $S_2$ 的交线在该点的切线的方程。

七、（8 分）试计算球台 $x^2 + y^2 + z^2 \leq 1$，$0 \leq z \leq 1/2$ 的表面积。
