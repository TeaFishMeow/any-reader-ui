## 试卷

### 一、简答题（每小题 6 分，共 48 分）

（1）已知 $\vec { \pmb { a } } = \{ \mathbf { 0 } , \mathbf { 1 } , 2 \} , \vec { \pmb { b } } = \{ \mathbf { 3 } , \mathbf { 4 } , \mathbf { 5 } \} , \vec { \pmb { c } } = \{ \mathbf { 6 } , 7 , \mathbf { 8 } \}$ ，求 $( { \overrightarrow { \mathbf { a } } } \times { \overrightarrow { \mathbf { b } } } ) \cdot { \overrightarrow { \mathbf { c } } }$ 。

（2） 设空间中四个点的坐标依次为 $A(1,0,0),\ B(0,1,0),\ C(0,0,1),\ D(-1,-1,-1)$。试证 $ABCD$ 是一个直角梯形，并计算该梯形的面积。

（3）已知直线 $L$ 的方程为 $\begin{cases} x + y - 1 = 0 \\ y - z + 1 = 0 \end{cases}$，平面 $\pi$ 的方程为 $x - y - \sqrt{3} = 0$，求过 $L$ 且与 $\pi$ 的夹角为 $60°$ 的平面的方程。

（4）设空间曲线的参数方程为 $x = 3\cos t,\ y = 2\sin t,\ z = t^2$，$0 \leq t \leq \pi$，求该曲线在 $xOy$ 平面上的投影。

（5）设隐函数 $z = f(x, y)$ 由 $xy^3 + yz^3 + zx^3 = 1$ 确定，求 $\partial z/\partial x$ 及 $\partial z/\partial y$，并计算这两个偏导数在点 $(1, 1, -1)$ 处的值。

（6）设 $z = uv + \sin(e^u + v)$，其中 $u = \ln(x^2 + y^2)$，$v = x^2 - y^2$。试用复合函数求导法则求 $\partial z/\partial x$ 及 $\partial z/\partial y$。

（7）计算二重积分： $\begin{array} { r } { I _ { 1 } = \iint _ { D } \sqrt { 1 - \left( x ^ { 2 } + y ^ { 2 } \right) } d x d y } \end{array}$ ， $\pmb { D }$ 由 $x ^ { 2 } + y ^ { 2 } = x$ 所围成。

（8）求曲面 $| x | ^ { 3 } + | y | ^ { 3 } + | z | ^ { 3 } = 3$ 在其上的点 $P ( \mathbf { 1 } , - \mathbf { 1 } , - \mathbf { 1 } )$ 处的切平面。

二、（10 分）求 $f ( x , y ) = x ^ { 3 } + y ^ { 3 } - 6 x y + 8$ 的极值点及相应的极值，并指出是极大值或极小值。

三、（12 分）已知曲面 $\Sigma$ 的方程为 $x^2 + y^2 - z^2 = 0$，试指出 $\Sigma$ 是哪一种二次曲面，求该曲面上与点 $A(1, 0, 2)$ 距离最短的点，并计算该最短距离。

四、（12 分）立体 $V$ 由曲面 $\Sigma: z = (x^2 + y^2)/2$ 及平面 $\pi: z = 4$ 所围成，计算：

（A） $I_2 = \iiint_V (x + y + z)\, dx\, dy\, dz$，（B）立体 $V$ 的表面积。

五、（8 分）曲面 $\Sigma$ 与平面 $\pi$ 由第四题定义，$\Sigma$ 与平面 $y = x$ 相交于一条曲线，此曲线被平面 $\pi$ 截出的有界部分为 $L$。试计算第一型曲线积分 $I_3 = \int_L \dfrac{1}{\sqrt{1+2z}}\, ds$。

六、（10 分）将椭圆形区域 $x^2 + 4y^2 \leq 1$ 的边界记为 $C$，试计算积分

$$
I_4 = \oint_C \left(y^2 - y + \ln(1+x^2)\right) dx + \left(e^{\sin y} + 2xy + x\right) dy
