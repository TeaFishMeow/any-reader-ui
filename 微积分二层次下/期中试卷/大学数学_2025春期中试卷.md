## 试卷

### 一、简答题（每小题6 分，共 48 分）

（1）设三个向量 $a = 2 i + j + k , b = 3 i - 2 j + 2 k , c = 4 j + 3 k$ ，求 ${ P r j } _ { c } ( { \pmb a } \times { \pmb b } )$ 。

（2）一条直线 $l$ 在平面 $x + 2y = 0$ 上，且和两直线 $l_1: \dfrac{x}{1} = \dfrac{y}{4} = \dfrac{z-1}{-1}$，$l_2: \dfrac{x-4}{2} = \dfrac{y-1}{0} = \dfrac{z-2}{-1}$ 都相交，求该直线 $l$ 的方程。

（3） 证明螺旋线 $x = a\cos t,\ y = a\sin t,\ z = bt$ 的切线与 $z$ 轴的夹角为定值。

（4）求空间曲线 $\Gamma: \begin{cases} x^2 + y^2 + z^2 = 9 \\ x + z = 1 \end{cases}$ 在各个坐标平面上的投影曲线方程。

（5）设隐函数 $z = z(x, y)$ 由方程 $\sin(xz) + \ln(z - y) = y + x^2$ 确定，求全微分 $dz(0, 0)$。

（6）求曲面 $x^2 + y^2 + z^2 - xy - 3 = 0$ 上同时垂直于平面 $z = 0$ 与 $x + y + 1 = 0$ 的切平面方程。

（7）计算二重积分： $\begin{array} { r } { I = \iint _ { D } \ | y - x ^ { 2 } | \ d x d y , } \end{array}$ , 其中 $\pmb { D }$ 为 $\mathbf { 0 } \leq x \leq 1 , \mathbf { 0 } \leq y \leq 1$ .

（8）求曲线积分 $\oint _ { L } \ ( x + y ) ^ { 2 } d s$ ，其中 $\pmb { L }$ 为圆周 $x ^ { 2 } + y ^ { 2 } = a ^ { 2 } ( a > 0 )$ 。

二、（10 分）设函数 $f ( x , y ) = x ^ { 3 } - 3 x ^ { 2 } - 2 y ^ { 2 }$ ，求函数的极值和函数在区域 $x ^ { 2 } + y ^ { 2 } \leq 1 6$ 上的最大小值。

三、（12 分）设 $f(x, y) = \begin{cases} x^{5/3} \sin\dfrac{y}{x}, & x \neq 0 \\ 0, & x = 0 \end{cases}$，(1) 求两偏导数 $f_x'(x, y),\ f_y'(x, y)$。(2) 判断函数 $f(x, y)$ 在 $(0, 0)$ 点处是否可微？

四、（12 分）立体 $\Omega$ 由曲面 $\Sigma_1: \dfrac{x^2 + y^2}{4} = 8 - z$ 及曲面 $\Sigma_2: z = \sqrt{x^2 + y^2}$ 所围成，计算：

（A）立体 $\Omega$ 的体积，（B）立体 $\Omega$ 的表面积。

五、（8 分）在从点 $O(0,0)$ 到点 $A(\pi, 0)$ 的曲线族 $y = a\sin x$（$a > 0$）中，求一条曲线 $L$，使得沿该曲线从 $O$ 到 $A$ 的曲线积分 $\int_L (1 + y^3)\, dx + (2x + y)\, dy$ 的值最小。

六、（10 分）设 $V$ 是由 $\begin{cases} x^2 = z \\ y = 0 \end{cases}$ 绕 $z$ 轴旋转一周生成的曲面与平面 $z = 2,\ z = 4$ 所围成的区域，试计算积分 $\iiint_V (x^2 + y^2 + z^2)\, dV$。
