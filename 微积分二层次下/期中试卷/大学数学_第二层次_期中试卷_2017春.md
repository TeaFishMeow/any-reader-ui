## 试卷

### 一、计算下列各题（每题6分，共48分）

1、 将函数 $\ln ( 1 - 2 x )$ 展开成 $x$ 的幂级数，并指出其收敛区间。

2、 已知 $( { \pmb a } \times { \pmb b } ) \cdot { \pmb c } = 2$ ，求 $\left[ ( a + b ) \times ( b + c ) \right] \cdot ( c + a ) \phantom { \left[ \frac { b + c } { b } \right] }$ 。

3、 设一平面经过原点及点 $( 6 , - 3 , 2 )$ ，且与平面 $4 x - y + 2 z = 8$ 垂直，求该平面的方程。

4、 求直线 $\left\{ { \begin{array} { l } { x - 2 y + 3 = 0 } \\ { 2 y + z - 7 = 0 } \end{array} } \right.$ 与平面 $3 x + 4 y + 5 z = 6$ 之间的距离。

5、 设函数 $f ( u , v )$ 具有二阶连续偏导数， $z = f ( x y , x ^ { 2 } - y ^ { 2 } )$ ，求 ${ \frac { \partial ^ { 2 } z } { \partial x \partial y } } .$ 0

6、 求椭圆抛物面 $x ^ { 2 } + y ^ { 2 } = z$ 在点 $( 1 , 1 , 2 )$ 的切平面。

7、 交换积分次序：$\int_0^1 dy \int_{-\sqrt{1-y^2}}^{1-y} f(x,y)\, dx$。

8、 求平面 $6 x + 3 y + 2 z - 6 = 0$ 被三个坐标平面所割出的部分的面积。

二、（10分）求幂级数 $\sum _ { n = 0 } ^ { \infty } ( 2 n + 1 ) x ^ { n }$ 的收敛域，并求其和函数。

三、（10 分）函数 $f(x, y)$ 在 $(0,0)$ 处连续，且极限 $\lim_{\substack{x\to 0\\y\to 0}} \dfrac{f(x,y)}{x^2+y^2}$ 存在，证明：$f(0,0) = 0$，$f_x'(0,0) = f_y'(0,0) = 0$，并讨论 $f(x,y)$ 在 $(0,0)$ 处的可微性。

四、（12分）求函数 $f ( x , y ) = x ^ { 4 } + y ^ { 4 } - x ^ { 2 } - 2 x y - y ^ { 2 }$ 的全部极值。

五、（10分）计算二重积分 $\iint _ { D } { \left| x ^ { 2 } + y ^ { 2 } - 1 \right| } d x d y ,$ 其中 $D = \{ ( x , y ) \mid 0 \leq x \leq 1 , \ 0 \leq y \leq 1 \}$ 。
